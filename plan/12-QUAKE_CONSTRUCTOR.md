# Quake Construct (Lua scripting for Q3)

A quake3 mod that allows anyone with Lua knowledge to create modifications without Quake3World touching C code. Scripts execute in realtime while the game is running — rapid prototyping. Hooks and bindings to the engine let you respond to in-game events. It can create shaders from Lua, spawn entities, draw 3D objects, and even has Bullet physics integration. This is the closest thing to what you need — a runtime scripting layer on top of Q3. The code is on Google Code (archived), mirrored on GitHub via QuakeEngines. Problem: abandoned ~2010, Windows only, and you're building a C layer not Lua. But the architecture and hook patterns are worth studying.

https://quakeconstruct.wordpress.com
