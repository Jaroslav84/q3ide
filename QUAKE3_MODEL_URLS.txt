MODEL FORMATS & SPECIFICATIONS
================================================================================

File Format: .pk3 (ZIP-compatible archive)
Alternative: .zip (extract to baseq3/)
Model Format: MD3 (Quake 3 native)

Standard Model Structure:
  models/players/<charactername>/
  ├── head.md3
  ├── lower.md3
  ├── upper.md3
  ├── model.shader
  └── texture.jpg/.tga

Level of Detail (LOD):
  - High: 800 faces
  - Medium: 500 faces
  - Low: 300 faces

Required Material Names:
  - l_legs (leg textures)
  - u_torso (torso textures)
  - h_head (head textures)

Installation Directory:
  /root/Projects/q3ide/baseq3/

INSTALLATION INSTRUCTIONS
================================================================================

1. Download model pack from one of the sources above

2. Extract to baseq3/ directory:
   cd /root/Projects/q3ide/baseq3/
   unzip model_pack.zip
   # OR
   cp model_pack.pk3 .

3. Verify installation:
   find . -path "*/models/players/*" -type d | head -10

4. Start game:
   cd /root/Projects/q3ide
   sh ./scripts/build.sh --run --level 0

5. In-game: Settings → Player → Model → [Select character]
